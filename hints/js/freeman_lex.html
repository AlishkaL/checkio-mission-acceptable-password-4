Since statement in return is true, when all items inside array are true, you just need to assign not required conditions to true, like they are already valid: 

if (password.length > 9) cond2 = cond3 = true;

Besides, since we may change values of cond2 and cond3, you need to change their type to var.

So how my final solution for this mission should look like?

Here it is: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length > 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length > 9) cond2 = cond3 = true; // cond 4
    
    return !([cond1, cond2, cond3].includes(false));
}
