Давай продовжимо розв'язок у підказках з попередньої місії в серії. 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(char.isdigit() for char in password)
    cond3 = any(char.isalpha() for char in password) 

    return all([cond1, cond2, cond3])
У цій місії тобі слід додати до розв'язку, що, при паролі довжиною більше 9, умови про наявність і кількість цифр не застосовуються.

То як мій остаточний розв'язок має виглядати?
Ось він: 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(char.isdigit() for char in password)
    cond3 = any(char.isalpha() for char in password)
    if len(password) > 9: # cond4    
        cond2 = cond3 = True

    return all([cond1, cond2, cond3])

Перевірка всього пароля на те, чи є він числом (текстове ціле число без букв) або всіма літерами (слово без цифр). Поєднання умов з допомогою дужок, операторів 'and' і 'or'. 

def is_acceptable_password(x: str) -> bool:

    return (len(x)>6 and not x.isdigit() and not x.isalpha()) or len(x)>9
Використання функції any() і каскаду порівнянь. 

def is_acceptable_password(x: str) -> bool:

    return any((len(x) > 9, 0 < sum(ch.isdigit() for ch in x) < len(x) > 6))
