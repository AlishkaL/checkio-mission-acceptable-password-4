我不明白，如何禁用条件。

由于return 中的语句是true ，当数组中的所有项目都是true ，你只需要将不需要的条件分配到true ，就像它们已经有效一样： 

if (password.length> 9) cond2 = cond3 = true;

此外，由于我们可能会改变cond2 和cond3 的值，你需要将它们的类型改为var 。

那么我的这个任务的最终解决方案应该是什么样子的呢？

这就是了： 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length> 9) cond2 = cond3 = true; // cond 4
    
    return!([cond1, cond2, cond3].includes(false));
}
